# zm-color-picker

## 介绍
ColorPicker 一个基于vue和canvas的颜色选择器。

### 安装
```
npm install zm-color-picker --save
# or 
yarn add zm-color-picker
```

### 引入
```
import Vue from 'vue';
import ZmColorPicker from 'zm-color-picker';
import "zm-color-picker/lib/zm-color-picker.css";

Vue.use(ZmColorPicker);
```
